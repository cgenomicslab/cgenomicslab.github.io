Teaching

Beyond our own research, we contribute to training across IMBB-FORTH and the University of Crete — from hands-on computational workshops to graduate-level coursework.

Courses & Workshops

IMBB Data Analysis Course

Institute-wide · Coordinated by CGLab

A hands-on introduction to Python programming, data analysis, and computational biology aimed at wet-lab biologists with no prior coding experience. The course explains the logic behind common analyses — not just how to run the tools — to build confidence and broaden computational literacy across the institute.

Week 1 covers the fundamentals: Python basics, data exploration and plotting, statistics and p-value logic, enrichment and expression analysis, and dimensionality reduction. Week 2 brings specialized sessions in genomics led by contributing labs, spanning bulk RNA-seq analysis, single-cell RNA-seq, ATAC-seq.

  • Python
  • Statistics
  • Expression analysis
  • Hands-on
